#QuickQuiz-M

可查看本地题库的在线答题软件

## 2018/1/30提交最初版本V0.1 Beta
目前实现的功能如下。

1. 各题库（包括单选、多选、判断、填空、简答、及各自定义类型）的浏览和答案查看。
2. 浏览时，按题目类型自定义学习范围。
3. 浏览时，打乱题目顺序，可依据随机Key来实现随机队列的复现。
4. 浏览时，跳转至题库中某一题目。
5. 浏览时，查找含有关键字的题目。
6. 各题库（包括单选、多选、判断、及各客观自定义类型）的测试和自动评分。
7. 夜间模式。
8. 题库数量容易扩展，理论支持一切由QuickQuiz 1.2.1 for Windows生成的qxml格式题库。
